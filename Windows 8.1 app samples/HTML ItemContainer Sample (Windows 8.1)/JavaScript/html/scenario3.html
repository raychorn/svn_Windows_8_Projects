The first item control on this page demonstrates how to insert interactive elements into an item container while allowing both the item container and the interactive element to behave as the user would expect. You'll notice the user can rate this item using the ratings control, while still being able to select the item elsewhere on it's surface.

The item control on the right demonstrates the ability to change the direction of selection for the item control to move left and right, as opposed to up and down. In this case up and down movement of the user scrolls, while left and right movement selects. This is useful when you need a selectable item on a vertically scrolling region.
