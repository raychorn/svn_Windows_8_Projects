Live Tiles are the representation of an application in Start and you can use tile and badge notifications to display new, relevant, and tailored content to the user.

Tile notifications are specified in an XML format and contain text and images in a pre-defined layout.

Applications must always include a square (square150x150) logo in the manifest. Similarly, if an application wants to use wide (wide310x150) templates, it MUST include a wide (wide310x150) logo with it as well. Applications that want to use square310x310 templates MUST include both wide310x150 and square310x310 logos. This pattern is STRONGLY RECOMMENDED since users decide what size your application's tile will be.
